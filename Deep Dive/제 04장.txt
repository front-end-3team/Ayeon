Deep Dive 제 04장 : 변수

4.1 변수란 무엇인가? 왜 필요한가?

    ** 변수 variable 이란?
    - 하나의 값을 저장하기 위해 확보한 메모리 공간 자체 또는 메모리 공간을 식별하기 위해 붙인 이름

    ** 변수 이름(변수명)이란?
    - 메모리 공간에 저장된 값을 식별할 수 있는 고유한 이름

    ** 할당 assignment (대입, 저장)이란?
    - 변수에 값을 저장하는 것

    ** 참조 reference 란?
    - 변수에 저장된 값을 읽어 들이는 것

4.2 식별자

    ** 식별자 identifier 란?
    - 어떤 값을 구별해서 식별할 수 있는 고유한 이름
    - 변수 이름을 식별자라고도 함
    - 식별자는 값이 아니라 메모리 주소를 기억
    - 식별자로 값을 구별해서 식별한다는 것은 식별자가 기억하고 있는 메모리 주소를 통해 메모리 공간에 저장된 값에 접근 가능하다는 의미

4.3 변수 선언
    - 변수 선언 variable declaration 이란 변수를 생성하는 것
    - 변수를 사용하기 위해서는 반드시 선언이 필요
    - 변수를 선언할 때 쓰는 키워드 : let const var(는 잘 안 씀)
    - 변수를 선언한 후, 값을 할당하지 않았으나 변수 선언에 의해 확보된 
      메모리 공간에는 자바스크립트 엔진에 의해 undefined라는 값이 암묵적으로 할당되어 초기화
    
    ** undefined : 자바스크립트에서 제공하는 원시 타입의 값 primitive value

4.4 변수 선언의 실행 시점과 변수 호이스팅
    - 변수 선언은 소스코드가 순차적으로 실행되는 시점인 런타임 이전에 먼저 실행
    
    ** 변수 호이스팅 variable hoisting 이란?
    - 변수 선언문이 코드의 선두로 끌어 올려진 것처럼 동작하는 자바스크립트 고유의 특징

4.5 값의 할당
    - 변수에 값을 할당할 때는 할당 연산자 = 를 사용
    - 값의 할당은 소스코드가 순차적으로 실행되는 시점인 런타임에 실행

4.6 값의 재할당
    
    ** 상수 constant 란?
    - 값을 재할당할 수 없어서 변수에 저장된 값을 변경할 수 없다면 상수라고 한다
    - 한번 정해지면 변하지 않는 값
